Part IV — The LUB Property

The LUB Axiom (Completeness of $\mathbb{R}$): Every nonempty set of real numbers that is bounded above has a least upper bound in $\mathbb{R}.$
$\forall S\in \mathcal{B},$ where $\mathcal{B}$ is the set of all nonempty bounded above subsets of $\mathbb{R},$ there exist $\alpha\in \mathbb{R}$ such that $lub(S)=\alpha.$
Why do we need an axiom?
$\mathbb{Q}$ does not satisfy this property. The set $S=\{q\in\mathbb{Q}\mid q^2<2\}$ is bounded above in $\mathbb{Q}$ (by $2$, say), but its least upper bound would be $\sqrt{2},$ and $\sqrt{2}\notin\mathbb{Q}.$ So no least upper bound of $S$ exists within $\mathbb{Q}$ — $\mathbb{Q}$ has gaps.

The Completeness Axiom is exactly what distinguishes $\mathbb{R}$ from $\mathbb{Q}$: in $\mathbb{R}$ there are no gaps.

Proof: LUB Property implies GLB Property

Theorem: If the Least Upper Bound Property holds, then the Greatest Lower Bound Property holds.

Proof:

  1. Let A be a nonempty set bounded below. Let L be the set of all lower bounds of A.
  2. L is nonempty because A is bounded below.
  3. L is bounded above: Any element $y \in A$ is an upper bound for L because $l \le y$ for all $l \in L$.
  4. By the LUB property, L has a least upper bound. Let $\alpha = lub(L)$.
  5. Show $\alpha$ is a lower bound for A: Assume $\alpha > x$ for some $x \in A$. Since $x$ is an upper bound for L (from step 3), and $\alpha$ is the least upper bound, we must have $\alpha \le x$. This is a contradiction. Thus, $\alpha \le a$ for all $a \in A$.
  6. Show $\alpha$ is greatest: Let m be any lower bound of A. Then $m \in L$. Since $\alpha$ is an upper bound for L, $m \le \alpha$.

Therefore, $\alpha = glb(A)$.

By using $lub(A)=-glb(-A),$ one may try an alternate proof.

Theorem — Archimedean Property (1) (AP1)
Given any real number $x \in \mathbb{R}$, there exists a natural number $n \in \mathbb{N}$ such that $n > x.$

Proof (using LUB Property): Assume, for contradiction, that $\mathbb{N}$ is bounded above, $1\in \mathbb{N}.$ That is $\mathbb{N}$ is bounded above and nonempty, hence by the Least Upper Bound Property, $s = lub(\mathbb{N})$ exists. Since $s$ is the least upper bound, $s-1$ is not an upper bound. Therefore, there exists an $m \in \mathbb{N}$ such that $m > s-1.$ Adding $1$ to both sides gives $m+1 > s.$ However, $m+1 \in \mathbb{N},$ which contradicts the assumption that $s$ is an upper bound for $\mathbb{N}.$ Hence our assumption $\mathbb{N}$ is bounded above is false. Equivalently $\mathbb{N}$ is not bounded above.

Theorem — Archimedean Property (2) & Equivalence
Statement: If $x, y \in \mathbb{R}$ with $x > 0,$ then there exists a natural number $n \in \mathbb{N}$ such that $nx > y.$

Proof of Equivalence:
  • (1) $\implies$ (2): Since $x > 0,$ the real number $\frac{y}{x}$ exists. By Archimedean Property (1), there exists $n \in \mathbb{N}$ such that $n > \frac{y}{x}.$ Multiplying by $x$ (which is positive) yields $nx > y.$
  • (2) $\implies$ (1): Let $x=1$ and let $y$ be any real number. Property (2) guarantees the existence of $n \in \mathbb{N}$ such that $n \cdot 1 > y,$ {i.e., $n > y.$ This proves Property (1).

Key Applications

The $1/n$ Rule
For any $\varepsilon>0,$ there exists $n\in\mathbb{N}$ such that $\dfrac{1}{n}<\varepsilon.$
Proof: Let $x>0.$ By Archimedian property, there exists $n\in \mathbb{N}$ such that $n>\frac{1}{x}.$ That is $x>\frac{1}{n}.$
This is the foundation of the $\varepsilon$-$\delta$ definition of limits.
Corollary — $\mathbb{Z}$ is Not Bounded Below
The set of integers $\mathbb{Z}$ has no lower bound in $\mathbb{R}.$ That is, for every $m\in\mathbb{R}$ there exists $z\in\mathbb{Z}$ such that $z < m.$
Proof — reducing to the Archimedean Property
Recall (Archimedean Property 1): $\mathbb{N}$ is not bounded above — for every $x\in\mathbb{R}$ there exists $n\in\mathbb{N}$ with $n > x.$

Claim: $\mathbb{Z}$ is not bounded below.

Let $m\in\mathbb{R}$ be arbitrary. We must find $z\in\mathbb{Z}$ with $z < m.$

Apply the Archimedean Property with $x := -m \in\mathbb{R}.$ There exists $n\in\mathbb{N}$ such that $$n \;>\; -m.$$ Multiplying both sides by $-1$ (reversing the inequality): $$-n \;<\; m.$$ Since $n\in\mathbb{N}\subseteq\mathbb{Z},$ we have $-n\in\mathbb{Z}$ and $-n < m.$

Since $m\in\mathbb{R}$ was arbitrary, no real number can serve as a lower bound for $\mathbb{Z}.$ Hence $\mathbb{Z}$ is not bounded below. $\blacksquare$
Another approach: Recall that if $A$ is bounded below, then $-A$ is bounded above. If $\mathbb{Z}$ is bounded below, then $\mathbb{Z}=-\mathbb{Z}$ is bounded above. But then $\mathbb{N}$ (as $\mathbb{N}\subset \mathbb{Z}$) is bounded above. A contradiction.
Why this matters
The fact that $\mathbb{Z}$ is not bounded below is used immediately in the next result: to prove that the set $S_x = \{z\in\mathbb{Z}\mid z\le x\}$ is nonempty for every real $x$ (since we can always find an integer strictly below $x$). This is a key step in the proof of the Floor / Integer Part Lemma.

Summary of the logical chain:
LUB Property  $\Longrightarrow$  $\mathbb{N}$ not bounded above  $\Longrightarrow$  $\mathbb{Z}$ not bounded below  $\Longrightarrow$  Floor Lemma  $\Longrightarrow$  Density of $\mathbb{Q}$ in $\mathbb{R}.$

Lemma 1 — Existence of Integer Part

For any real number $ x $, there exists an integer $ m$ such that:

$ m \le x < m+1 $
Proof: Define the set $ S_x = \{ z \in \mathbb{Z} \mid z \le x \}.$
  1. $ S_x $ is non-empty: Since $\mathbb{Z}$ is not bounded below (Archimedean property), there exists an integer $ k $ such that $k < x $, so $ k \in S_x $.
  2. $ S_x $ is bounded above: By definition of $S_x$, $ x$ is an upper bound for $ S_x $.
  3. By the Least Upper Bound (LUB) property, let $m = lub(S_x) $.
  4. Since $ m-1$ is not an upper bound for $ S_x $, there exists some $k \in S_x $ such that $k > m-1.$
    • This implies $ k+1 > m.$
    • Since $ m $ is the $lub(S_x)$, $k+1 \notin S_x $, which means $k+1 > x $.
    • Since $k \le x $ (as $k \in S_x ), m $ must be an integer trapped between $k$ and $k+1$. Thus $m = k$.
    Therefore, $m \le x < m+1.$
Figure 1: The set $ S_x $ contains all integers $ \le x $ (e.g., $ k, m $). The supremum $ m $ is the largest integer less than or equal to $ x $.
Theorem — Uniqueness of the Integer Part

The integer $ m $ satisfying $ m \le x < m+1 $ is unique.

Proof: Assume for contradiction that there are two distinct integers $ m $ and $ n $ such that:
$ m \le x < m+1 \quad \text{and} \quad n \le x < n+1 $
Without loss of generality, assume $ n > m $. Since $ m $ and $ n $ are integers, the next integer after $ m $ is $ m+1 $. Therefore: $ n \ge m+1 $ However, applying the property of $ x $ with respect to $ n $ and $ m $: $ n \le x \quad \text{and} \quad x < m+1 $ Combining these inequalities yields: $ n \le x < m+1 \implies n < m+1.$ We have arrived at a contradiction: $ n $ cannot be both $ \ge m+1 $ and $ < m+1 $. Thus, our assumption that $ m \neq n $ is false. Therefore, the integer $ m $ is unique.
Figure 2: If two distinct integers $ m $ and $ n $ (where $ n > m $) both satisfied the condition, their corresponding intervals $[m, m+1)$ and $[n, n+1)$ would have to contain the same point $x$. This is impossible because integer intervals are disjoint.

Motivation for the Density of Rationals

Step 0 — Integers between real numbers: sufficient but not necessary

Before asking about rationals, let us first think about integers between two real numbers.

Sufficient condition: If $b - a > 1,$ then there is always at least one integer strictly between $a$ and $b.$

Why? Let $m = \lfloor a \rfloor + 1$ (the integer just above $a$). Then $m > a.$ Since $b > a + 1 > m,$ we also have $m < b.$ So $a < m < b.$ ✓

Not necessary: The condition $b - a > 1$ is not necessary — an integer can lie between $a$ and $b$ even when $b - a \le 1.$

$0.2$$0.8$$0.6 \le 1$ No integer in $(0.2, 0.8)$ Condition $b-a > 1$ is not necessary

Conclusion: Whether an integer falls between $a$ and $b$ depends on the position of $a$ and $b$ relative to $\mathbb{Z}$, not merely on their distance.
Step 1 — Reformulating: rational between $a$ and $b$ $\Longleftrightarrow$ integer between $na$ and $nb$

We want to find a rational $\dfrac{m}{n}$ (with $n\in\mathbb{N},\, m\in\mathbb{Z}$) satisfying $a < \dfrac{m}{n} < b.$

Multiplying through by $n > 0$, this is equivalent to: $$na \;<\; m \;<\; nb.$$ So finding a rational between $a$ and $b$ is exactly the same as finding an integer $m$ between $na$ and $nb.$

Now the question becomes: can we choose $n$ large enough so that an integer is guaranteed to lie between $na$ and $nb$?

From the sufficient condition above, it is enough to have $nb - na > 1,$ i.e., $$n(b - a) > 1 \quad\Longleftrightarrow\quad n > \frac{1}{b-a}.$$ Since $b > a,$ we have $b - a > 0,$ so $\dfrac{1}{b-a}$ is a finite positive real number. The Archimedean Property guarantees that such an $n\in\mathbb{N}$ exists!
Step 2 — The plan of the proof
  1. Use the Archimedean Property to find $n\in\mathbb{N}$ with $n(b-a) > 1,$ i.e., $nb - na > 1.$
  2. Use the Floor / Integer Part Lemma to find an integer $m$ with $m \le na < m+1,$ i.e., $m$ is the integer part of $na.$
  3. Show that this $m+1$ satisfies $na < m+1 < nb,$ i.e., $\dfrac{m+1}{n}$ lies strictly between $a$ and $b.$

Theorem — Density of Rationals in $\mathbb{R}$
If $x, y \in \mathbb{R}$ with $x < y,$ there exists a rational number $r \in \mathbb{Q}$ such that $x < r < y.$

Proof: Since $y - x > 0,$ by the Archimedean Property (2), there exists $n \in \mathbb{N}$ such that $n(y-x) > 1,$ which implies $nx + 1 < ny.$
By the integer part property (Concept 3), there exists an integer $m$ such that $m \le nx < m+1.$
From $nx < m+1,$ we have $x < \frac{m+1}{n}.$
From $m \le nx,$ we have $m+1 \le nx + 1.$ Since $nx + 1 < ny,$ it follows that $m+1 < ny,$ which implies $\frac{m+1}{n} < y.$
Thus, choosing $r = \frac{m+1}{n},$ we have $x < r < y.$
Theorem — Density of Irrationals in $\mathbb{R}$
If $a, b \in \mathbb{R}$ with $a < b,$ there exists an irrational number $\xi \in \mathbb{R}\setminus\mathbb{Q}$ such that $a < \xi < b.$
Key idea — shift by $\sqrt{2}$ to convert the problem

We want an irrational between $a$ and $b.$ The trick is to shift both endpoints by $-\sqrt{2}$ and apply the Density of Rationals to the shifted interval.

Consider the two real numbers $a + \sqrt{2}$ and $b + \sqrt{2}.$ Since $a < b,$ we have $a + \sqrt{2} < b + \sqrt{2}.$

By the Density of Rationals, there exists a rational number $r\in\mathbb{Q}$ such that $$a + \sqrt{2} \;<\; r \;<\; b + \sqrt{2}.$$ Subtracting $\sqrt{2}$ throughout: $$a \;<\; r - \sqrt{2} \;<\; b.$$ Set $\xi = r - \sqrt{2}.$ Then $a < \xi < b$ and $\xi$ lies strictly between $a$ and $b.$ ✓
Proof — $\xi = r - \sqrt{2}$ is irrational
Suppose for contradiction that $\xi = r - \sqrt{2} \in \mathbb{Q}.$ Then $\sqrt{2} = r - \xi$ would be a difference of two rationals, hence rational. But $\sqrt{2}\notin\mathbb{Q}$ — contradiction.

Therefore $\xi = r - \sqrt{2}$ is irrational, and $a < \xi < b.$ $\blacksquare$

Why $\sqrt{2}$? Could we use any irrational?
Yes — any fixed irrational $\alpha$ works in place of $\sqrt{2}.$ The argument is:
  1. Apply density of rationals to $(a+\alpha, b+\alpha)$ to get $r\in\mathbb{Q}$ with $a+\alpha < r < b+\alpha.$
  2. Then $a < r-\alpha < b.$
  3. $r - \alpha$ is irrational because if $r - \alpha \in\mathbb{Q}$ then $\alpha = r-(r-\alpha)$ would be rational — contradiction.
$\sqrt{2}$ is the most classical choice.

Summary: Between any two real numbers there is both a rational and an irrational number. In fact, between any two reals there are infinitely many of each, which follows by applying the two density theorems repeatedly.
Q: Find glb of $\{x + x^{-1} \mid x > 0\}$. Is the set bounded above?
Let $S = \left\{x + \frac{1}{x} \;\middle|\; x > 0\right\}.$ We want to find the greatest lower bound and determine if the set is bounded above.

1. Greatest Lower Bound (glb): Since $x > 0$, both $x$ and $\frac{1}{x}$ are strictly positive. We can apply the Arithmetic Mean-Geometric Mean (AM-GM) Inequality, which states that for any $a, b > 0$, $\frac{a+b}{2} \ge \sqrt{ab}$.
Let $a = x$ and $b = \frac{1}{x}.$ Then: $$ \frac{x + \frac{1}{x}}{2} \ge \sqrt{x \cdot \frac{1}{x}} = \sqrt{1} = 1 $$ Multiplying by 2, we get: $$ x + \frac{1}{x} \ge 2 $$ This shows that $2$ is a lower bound for the set $S.$

To show that $2$ is the greatest lower bound, we must show that $2 \in S.$ If we choose $x = 1$ (which is $> 0$), we get: $$ 1 + \frac{1}{1} = 2 \in S $$ Since the lower bound $2$ is actually an element of the set, it is the minimum of the set, and therefore the greatest lower bound. $$ \text{glb}(S) = 2 $$
2. Is the set bounded above? No, the set is not bounded above.

To see this, consider what happens as $x$ approaches $0$ from the right. Let $x = \frac{1}{n}$ for a large natural number $n.$ $$ x + \frac{1}{x} = \frac{1}{n} + n $$ As $n \to \infty$ (which means $x \to 0^+$), the term $n$ grows arbitrarily large.
By the Archimedean Property, for any real number $M > 0$, we can choose $n > M,$ which means $\frac{1}{n} + n > M.$ Therefore, no real number can serve as an upper bound for $S.$

The Nested Interval Theorem

Theorem — Nested Interval Theorem (via LUB Property)
For each $n\in\mathbb{N},$ let $I_n = [a_n,\, b_n]$ be a closed bounded interval. Suppose the intervals are nested, meaning $$I_1 \;\supseteq\; I_2 \;\supseteq\; I_3 \;\supseteq\;\cdots, \quad\text{i.e.,}\quad a_n \le a_{n+1} \;\text{ and }\; b_{n+1} \le b_n \;\text{ for all }n.$$ Then:
  1. The intersection $\displaystyle\bigcap_{n=1}^{\infty} I_n$ is nonempty; in fact $$\bigcap_{n=1}^{\infty} [a_n, b_n] \;=\; [\,\text{lub}\{a_n\},\;\text{glb}\{b_n\}\,].$$
  2. If in addition the lengths become arbitrarily small (i.e., for every $\epsilon > 0$ there exists $n$ such that $b_n - a_n < \epsilon$), then the intersection contains exactly one point: $$\bigcap_{n=1}^{\infty} [a_n, b_n] \;=\; \{\,c\,\}, \quad\text{where}\quad c = \text{lub}\{a_n\} = \text{glb}\{b_n\}.$$
Proof (Using Only LUB/GLB Properties)
Step 1 — Define the sets of endpoints.

Let $A = \{ a_n \mid n \in \mathbb{N} \}$ be the set of all left endpoints. Let $B = \{ b_n \mid n \in \mathbb{N} \}$ be the set of all right endpoints.

Step 2 — Apply LUB and GLB.

Since the intervals are nested ($a_n \le a_{n+1}$), every $b_k$ (e.g. $b_1$) is an upper bound for $A$. By the LUB Property, let $\alpha = \text{lub}(A).$

Since the intervals are nested ($b_n \ge b_{n+1}$), every $a_k$ (e.g. $a_1$) is a lower bound for $B$. By the GLB Property (which follows from LUB), let $\beta = \text{glb}(B).$

Step 3 — Prove $\alpha \le \beta.$

We must show that the least upper bound of the lefts does not exceed the greatest lower bound of the rights.
For any $m, n \in \mathbb{N}$, consider $I_m = [a_m, b_m]$ and $I_n = [a_n, b_n]$. Let $k = \max(m, n)$. By nesting, $I_k \subseteq I_m$ and $I_k \subseteq I_n$. This implies $a_m \le a_k \le b_k \le b_n.$

Thus, for any fixed $n$, $a_n \le b_m$ for all $m$. This means $a_n$ is a lower bound for the set $B$. Since $\beta = \text{glb}(B)$ is the greatest lower bound, we must have $a_n \le \beta.$
Since this holds for all $a_n \in A$, $\beta$ is an upper bound for $A$. Since $\alpha = \text{lub}(A)$ is the least upper bound, we conclude $\alpha \le \beta.$

Step 4 — Identify the intersection as $[\alpha, \beta].$

($\supseteq$) Let $x \in [\alpha, \beta].$ For any $k$, we have $a_k \le \alpha \le x \le \beta \le b_k.$ Thus $x \in I_k$ for all $k$, so $x \in \bigcap I_k.$

($\subseteq$) Let $x \in \bigcap I_k.$ Then $a_k \le x \le b_k$ for all $k.$ Since $x$ is an upper bound for $A$, $\alpha \le x.$ Since $x$ is a lower bound for $B$, $x \le \beta.$ Thus $x \in [\alpha, \beta].$

Therefore, $\bigcap I_k = [\alpha, \beta] \neq \emptyset.$

Step 5 — Uniqueness (Lengths $\to 0$).

Suppose for every $\epsilon > 0$, there exists $n$ such that $b_n - a_n < \epsilon.$ Since $a_n \le \alpha \le \beta \le b_n$, we have $0 \le \beta - \alpha \le b_n - a_n.$ Thus $0 \le \beta - \alpha < \epsilon$ for any $\epsilon > 0.$ The only non-negative number smaller than all positive numbers is $0$. Hence $\beta - \alpha = 0$, so $\alpha = \beta = c.$ The intersection is the singleton $\{c\}.$ $\blacksquare$
Why "closed" and "bounded" are essential
The theorem fails for open intervals and fails if unbounded.

Counterexample 1 (Open): $I_n = \left(0,\,\dfrac{1}{n}\right).$ These are nested and bounded, yet $\displaystyle\bigcap_{n=1}^\infty\!\left(0,\tfrac{1}{n}\right) = \emptyset.$ (If $x > 0$, pick $n > 1/x$, then $x \notin I_n$.)

Counterexample 2 (Unbounded): $I_n = [n, \infty).$ These are nested and closed, yet $\bigcap_n [n, \infty) = \emptyset.$

Self-Assessment: Applications of the Archimedean Property

Test your understanding of the Archimedean Property and interval properties.

Question 1

Consider the sequence of intervals $J_n = \left(0, \frac{1}{n}\right).$ What is the intersection $\bigcap_{n=1}^{\infty} J_n$?

Show Hint
Assume there is a point $x$ in the intersection. Then $x$ must be in every interval. This implies $0 < x < \frac{1}{n}$ for all $n \in \mathbb{N}$. Use the Archimedean Property to show this is impossible.
Show Solution
Answer: $\emptyset$ (Empty Set).

Proof: Suppose, for contradiction, that there exists $x \in \bigcap_{n=1}^{\infty} J_n.$ Then $x > 0$ and $x < \frac{1}{n}$ for all $n \in \mathbb{N}.$ Since $x > 0$, $\frac{1}{x}$ is a real number. By the Archimedean Property, there exists an $N \in \mathbb{N}$ such that $N > \frac{1}{x}.$ This implies $\frac{1}{N} < x.$ But this contradicts the requirement that $x < \frac{1}{n}$ for all $n$. Therefore, no such $x$ exists, and the intersection is empty.
Question 2

Consider the sequence of intervals $J_n = \left(-\frac{1}{n}, \frac{1}{n}\right).$ What is the intersection $\bigcap_{n=1}^{\infty} J_n$?

Show Hint
Is 0 in every interval? What about any other number $x \neq 0$? If $x \neq 0$, then $|x| > 0$. Apply the Archimedean Property to $|x|$.
Show Solution
Answer: $\{0\}.$

Proof:
  1. For any $n$, $-\frac{1}{n} < 0 < \frac{1}{n}$, so $0 \in J_n$ for all $n$. Thus $0 \in \bigcap J_n.$
  2. Let $x \neq 0$ be any other real number. Then $|x| > 0.$ By the Archimedean Property, there exists $N \in \mathbb{N}$ such that $N > \frac{1}{|x|}.$ This implies $\frac{1}{N} < |x|.$ This means $x > \frac{1}{N}$ or $x < -\frac{1}{N}$. In either case, $x \notin \left(-\frac{1}{N}, \frac{1}{N}\right).$
Since no non-zero number is in the intersection, $\bigcap J_n = \{0\}.$
Question 3

Prove that the real numbers can be written as a disjoint union: $$ \mathbb{R} = \bigcup_{z \in \mathbb{Z}} (z, z+1] $$ (i.e., show that every real number falls into exactly one such interval).

Show Hint
Use the Existence of the Integer Part lemma: For any $x \in \mathbb{R}$, there exists a unique integer $m$ such that $m \le x < m+1.$ Check the cases where $x$ is an integer vs. non-integer.
Show Solution
Proof: Let $x \in \mathbb{R}$ be arbitrary.

1. Existence (Coverage): By the Integer Part Lemma, there exists an integer $m$ such that $m \le x < m+1.$
  • If $x$ is not an integer, then $m < x < m+1$, so $x \in (m, m+1].$
  • If $x$ is an integer, then $x = m$. Since $m \le m < m+1$, we have $x \in (m-1, m].$
In both cases, $x$ belongs to the union.

2. Disjointness: Suppose $x \in (z, z+1] \cap (z', z'+1]$ for some integers $z, z'.$ Without loss of generality, assume $z < z'.$ Then $z' \ge z+1$ (since integers are discrete). We have $z < x \le z+1$ (from first interval) and $z' < x \le z'+1$ (from second). Combining these: $z' < x \le z+1 \le z'.$ This implies $z' < z'$, which is a contradiction. Thus, the intervals are pairwise disjoint.
