The Modulus (Absolute Value) Function

A complete lecture covering definition · properties · proofs · geometry · boundedness · applications — with interactive explorers and 24 self-assessment problems.

Section 1

Definition

Definition — Absolute Value / Modulus For $a\in\mathbb{R},$ the absolute value (or modulus) of $a,$ written $|a|,$ is defined as the larger of $a$ and $-a$: $$|a| \;=\; \max\{a,\,-a\}.$$ Geometrically, $|a|$ is the distance from the point $a$ to the origin $0$ on the real number line.

Distance is always non-negative, motivating $|a|\ge 0.$ The piecewise form follows at once:

$$|a| = \begin{cases} a & \text{if } a\ge 0,\\ -a & \text{if } a < 0. \end{cases}$$
Proof of the piecewise form. Case 1 ($a\ge 0$): Then $-a\le 0\le a,$ so $\max\{a,-a\}=a.$ ✓
Case 2 ($a<0$): Then $-a>0>a,$ so $\max\{a,-a\}=-a.$ ✓
$\blacksquare$
Key immediate facts
  • $|a|\ge 0$ for all $a\in\mathbb{R}$, with $|a|=0\iff a=0.$
  • $|-a|=|a|$  (symmetry / even function).
  • $-|a|\le a\le|a|$  (the sandwich lemma).
  • $\sqrt{a^2}=|a|$ — a square-root identity used constantly in proofs.
Section 2

Geometric Interpretation & the V-Graph

Distance on the Real Line
  • $|x|$ = distance from $x$ to $0.$
  • $|x-y|$ = distance from $x$ to $y.$
  • $|x-c|<\delta \;\iff\; c-\delta < x < c+\delta$ — an open interval of radius $\delta$ centred at $c$ (a neighbourhood).

The graph is a "V-shape" with the vertex at the origin (0,0). The function is continuous everywhere but not differentiable at $x = 0$ due to the sharp corner.

For $x \ge 0$ The graph follows $y = x$ (slope $+1$). The function is increasing.
For $x < 0$ The graph follows $y = -x$ (slope $-1$). The function is decreasing.

Inductive step: Assume $|a^k|=|a|^k.$ Then $|a^{k+1}|=|a^k\cdot a|=|a^k||a|=|a|^k\cdot|a|=|a|^{k+1}.$ ✓
$\blacksquare$
Common Pitfalls
  • $|a+b|\ne|a|+|b|$ in general — equality holds only when $a$ and $b$ have the same sign (or one is zero).
  • $|a-b|\ne|a|-|b|$ in general.
  • $\sqrt{a^2+b^2}\ne|a|+|b|$ in general (this is for the 2-D modulus).
Section 4

Characterisation via Inequalities

For any $c>0$ and $x,a\in\mathbb{R},$ the following equivalences connect absolute value to interval notation:

$$|x| \le c \;\iff\; -c \le x \le c$$ $$|x| < c \;\iff\; -c < x < c$$ $$|x| \ge c \;\iff\; x\le -c \;\text{ or }\; x\ge c$$ $$|x-a| < \delta \;\iff\; a-\delta < x < a+\delta \quad\text{(neighbourhood of }a\text{)}$$
Proof of $|x|\le c \iff -c\le x\le c.$ ($\Rightarrow$): Suppose $|x|\le c.$ Since $|x|\ge x,$ we get $x\le c.$ Since $|x|\ge -x,$ we get $-x\le c,$ i.e., $-c\le x.$ So $-c\le x\le c.$

($\Leftarrow$): Suppose $-c\le x\le c.$ Then $x\le c$ and $-x\le c.$ Since $|x|=\max\{x,-x\},$ we get $|x|\le c.$
$\blacksquare$
Application — $\varepsilon$-$\delta$ and $\varepsilon$-$N$ Language The statement "$|x_n - L|<\varepsilon$ for all $n\ge N$" is precisely the formal definition of $x_n\to L.$ Writing $|x_n-L|<\varepsilon$ is shorthand for $L-\varepsilon < x_n < L+\varepsilon$. Every limit and continuity argument in analysis hinges on this characterisation.
Theorem — Multiplicative Property of Absolute Value (Second Proof)
Theorem: For any $a, b \in \mathbb{R}$, $|ab| = |a||b|.$

Given Definition: For any $x \in \mathbb{R}$, the absolute value is defined as $|x| = \max\{x, -x\}.$
This directly implies the standard piecewise definition: $$ |x| = \begin{cases} x & \text{if } x \ge 0 \\ -x & \text{if } x < 0 \end{cases} $$
Proof: By the definition of absolute value, evaluating $|ab|$ yields exactly one of two possibilities. We proceed by analyzing these two mutually exclusive cases.

Case 1: $|ab| = ab$
If $|ab| = ab$, then by the definition of absolute value, it must be that $ab \ge 0.$
For the product of two real numbers to be non-negative ($ab \ge 0$), the numbers must have the same sign. We split this into two subcases:
  1. Subcase 1a: $a \ge 0$ and $b \ge 0$
    Since $a \ge 0$, the definition gives $|a| = a.$
    Since $b \ge 0$, the definition gives $|b| = b.$
    Multiplying these together: $$ |a||b| = (a)(b) = ab $$ Since we are in the case where $|ab| = ab$, we have $|a||b| = |ab|.$
  2. Subcase 1b: $a < 0$ and $b < 0$
    Since $a < 0$, the definition gives $|a| = -a.$
    Since $b < 0$, the definition gives $|b| = -b.$
    Multiplying these together: $$ |a||b| = (-a)(-b) = ab $$ (The negatives cancel out). Again, since we are in the case where $|ab| = ab$, we have $|a||b| = |ab|.$


Case 2: $|ab| = -ab$
If $|ab| = -ab$, then by the definition of absolute value, it must be that $ab < 0.$
For the product of two real numbers to be negative ($ab < 0$), the numbers must have opposite signs. We split this into two subcases:
  1. Subcase 2a: $a \ge 0$ and $b < 0$
    Since $a \ge 0$, the definition gives $|a| = a.$
    Since $b < 0$, the definition gives $|b| = -b.$
    Multiplying these together: $$ |a||b| = (a)(-b) = -ab $$ Since we are in the case where $|ab| = -ab$, we have $|a||b| = |ab|.$
  2. Subcase 2b: $a < 0$ and $b \ge 0$
    Since $a < 0$, the definition gives $|a| = -a.$
    Since $b \ge 0$, the definition gives $|b| = b.$
    Multiplying these together: $$ |a||b| = (-a)(b) = -ab $$ Since we are in the case where $|ab| = -ab$, we have $|a||b| = |ab|.$

Conclusion: In all possible subcases arising from $|ab| = ab$ or $|ab| = -ab$, we arrived at $|a||b| = |ab|.$ Therefore, the equality holds for all $a, b \in \mathbb{R}.$ $\blacksquare$
Section 5

The Triangle Inequality

Theorem (Triangle Inequality) For all $a,b\in\mathbb{R}:$ $$|a+b|\le|a|+|b|.$$ Substituting $b\mapsto -b$ also gives $|a-b|\le|a|+|b|.$
Proof 1(1) — via the sandwich. From the sandwich lemma: $-|a|\le a\le|a|$ and $-|b|\le b\le|b|.$ Adding these: $$-(|a|+|b|)\le a+b\le|a|+|b|.$$ By the characterisation ($|x|\le c\iff -c\le x\le c$) with $c=|a|+|b|:$ $$|a+b|\le|a|+|b|.$$
$\blacksquare$
Proof 1(2) — via the sandwich (little modification to above). Since $a\le |a|, b\le |b|,$ we have $a+b\le |a|+|b|.$
Since $-a\le |a|, -b\le |b|,$ we have $-(a+b)\le |a|+|b|.$
Hence $\max\{a+b,-(a+b)\}=|a+b|\le |a|+|b|.$ Thus $$|a+b|\le|a|+|b|.$$
$\blacksquare$
Proof 2 — via squaring. $$|a+b|^2=(a+b)^2=a^2+2ab+b^2.$$ Since $ab\le|a||b|$ (as $ab\le|ab|=|a||b|$): $$a^2+2ab+b^2\le|a|^2+2|a||b|+|b|^2=(|a|+|b|)^2.$$ Both sides are non-negative; taking square roots gives $|a+b|\le|a|+|b|.$
$\blacksquare$

5.1 Generalised Triangle Inequality

Theorem For $a_1,\ldots,a_n\in\mathbb{R}:$ $$\left|\sum_{k=1}^{n}a_k\right|\le\sum_{k=1}^{n}|a_k|.$$
Proof by induction. Base ($n=2$): the Triangle Inequality. ✓
Step: Let $S=a_1+\cdots+a_k.$ Assume $|S|\le\sum_{i=1}^k|a_i|.$ Then $|S+a_{k+1}|\le|S|+|a_{k+1}|\le\sum_{i=1}^{k+1}|a_i|.$
$\blacksquare$
Section 6

The Reverse Triangle Inequality

Theorem (Reverse Triangle Inequality) For all $a,b\in\mathbb{R}:$ $$\bigl||a|-|b|\bigr|\le|a-b|.$$
Proof. Apply the Triangle Inequality to $a=(a-b)+b$: $$|a|\le|a-b|+|b|\;\Rightarrow\;|a|-|b|\le|a-b|.\tag{1}$$ Apply it to $b=(b-a)+a$: $$|b|\le|b-a|+|a|=|a-b|+|a|\;\Rightarrow\;|b|-|a|\le|a-b|.\tag{2}$$ Combining (1) and (2): $-|a-b|\le|a|-|b|\le|a-b|,$ which gives $\bigl||a|-|b|\bigr|\le|a-b|.$
$\blacksquare$
Why the Reverse TI is essential in analysis It converts a bound on $|x_n-x|$ into a bound on $\bigl||x_n|-|x|\bigr|$: $$\bigl||x_n|-|x|\bigr|\le|x_n-x|<\varepsilon$$ — proving that $x_n\to x\;\Rightarrow\;|x_n|\to|x|$ (continuity of the modulus function).
Geometric Intuition On the number line, $|a-b|$ is the distance between $a$ and $b$. The Reverse TI says: the difference of the distances from the origin ($|a|$ and $|b|$) can never exceed the distance between $a$ and $b$. Think of it as: "you cannot gain distance to the origin faster than you move."
Section 7

Absolute Value & Boundedness

Theorem — Characterisation of Boundedness Let $A\subseteq\mathbb{R}.$ Then $A$ is bounded (bounded above and below) if and only if $$\exists\,M>0:\;\forall x\in A,\;|x|\le M.$$
Proof ($\Rightarrow$). Suppose $A$ is bounded below by $L$ and above by $U.$ Set $M=\max(|L|,|U|).$ For all $x\in A$: $-M\le L\le x\le U\le M,$ so $|x|\le M.$
$\blacksquare$
Proof ($\Leftarrow$). If $|x|\le M$ for all $x\in A,$ then $-M\le x\le M,$ so $-M$ is a lower bound and $M$ is an upper bound. Hence $A$ is bounded.
$\blacksquare$
Worked Example $A=(-4,7].$  Choose $M=\max(|-4|,|7|)=7.$ For all $x\in A$: $-7\le -4<x\le 7,$ so $|x|\le 7.$ ✓

The smallest valid $M$ is exactly $\sup_{x\in A}|x|=7.$
Section 8

Worked Exercises

Exercise A — Solving $|2x-3|<5$

Solution. By the characterisation: $-5<2x-3<5.$ $$-5+3<2x<5+3\;\Rightarrow\;-2<2x<8\;\Rightarrow\;-1<x<4.$$ Solution set: $x\in(-1,4).$
$\blacksquare$

Exercise B — Solving $|3x-2|=|5x+4|$

Solution. Use $|a|=|b|\iff a=\pm b.$

Case 1 ($a=b$): $3x-2=5x+4\Rightarrow -2x=6\Rightarrow x=-3.$
Case 2 ($a=-b$): $3x-2=-(5x+4)\Rightarrow 8x=−2\Rightarrow x=-\tfrac{1}{4}.$

Solution set: $\left\{-3,\,-\tfrac{1}{4}\right\}.$
$\blacksquare$

Exercise C — Solving $|x+1|\ge 3$

Solution. $|x+1|\ge 3\iff x+1\le -3$ or $x+1\ge 3.$
Case 1: $x\le -4.$   Case 2: $x\ge 2.$

Solution set: $(-\infty,-4]\cup[2,+\infty).$
$\blacksquare$

Exercise D — Bounding $|x^2-9|$ near $x=3$

Solution: Show $|x-3|<\tfrac12\Rightarrow|x^2-9|<\tfrac{15}{2}.$ Factor: $|x^2-9|=|x-3||x+3|.$
From $|x-3|<\tfrac12$: $\tfrac52<x<\tfrac72,$ so $|x+3|<\tfrac72+3=\tfrac{13}{2}.$
Then $|x^2-9|<\tfrac12\cdot\tfrac{13}{2}=\tfrac{13}{4}<\tfrac{15}{2}.$
$\blacksquare$
Self-Assessment — Part A

9. Computational & Proof Skills (Q1–Q6)

Q1. Solve $|4x-5|\le 11.$ Express the answer as an interval.

Solution. $-11\le 4x-5\le 11\;\Rightarrow\;-6\le 4x\le 16\;\Rightarrow\;-\tfrac{3}{2}\le x\le 4.$
Answer: $x\in\!\left[-\tfrac{3}{2},\,4\right].$

Q2. Prove that $|x|-|y|\le|x-y|$ for all $x,y\in\mathbb{R}.$

Proof. Write $x=(x-y)+y$ and apply the Triangle Inequality: $|x|\le|x-y|+|y|.$ Subtract $|y|$: $|x|-|y|\le|x-y|.$ $\blacksquare$

Q3. Prove $|a+b|\le|a|+|b|$ without using squaring.

Proof. We have $-|a|\le a\le|a|$ and $-|b|\le b\le|b|.$ Adding: $-(|a|+|b|)\le a+b\le|a|+|b|.$ By the characterisation of absolute value, $|a+b|\le|a|+|b|.$ $\blacksquare$

Q4. For which $x\in\mathbb{R}$ does $|x^2-4|<3$?

Solution. $|x^2-4|<3\iff -3<x^2-4<3\iff 1<x^2<7.$
$x^2>1\iff|x|>1\iff x<-1$ or $x>1.$
$x^2<7\iff|x|<\sqrt7\iff -\sqrt7<x<\sqrt7.$
Intersection: $x\in(-\sqrt7,-1)\cup(1,\sqrt7).$

Q5. Prove: $|a^2-b^2|\le(|a|+|b|)|a-b|.$

Proof. $|a^2-b^2|=|(a-b)(a+b)|=|a-b||a+b|\le|a-b|(|a|+|b|)$ by the Triangle Inequality applied to $|a+b|.$ $\blacksquare$

Q6. Let $A=[-3,5)\cup\{7\}.$ Find the smallest $M>0$ with $|x|\le M$ for all $x\in A.$

Solution. Elements of $A$ satisfy $-3\le x<5$ or $x=7.$ So $|x|\le\max(|-3|,|7|)=7.$ The smallest valid $M$ is $M=7.$
Self-Assessment — Part B

10. Deeper Results (Q7–Q13)

Q7. Show that for all $a,b,c\in\mathbb{R}$: $|a-c|\le|a-b|+|b-c|.$ Interpret this geometrically.

Proof. Write $a-c=(a-b)+(b-c)$ and apply the Triangle Inequality: $|a-c|\le|a-b|+|b-c|.$ $\blacksquare$

Geometric: The distance from $a$ to $c$ is at most the sum of distances from $a$ to $b$ and from $b$ to $c$ — the triangle inequality for points on a line.

Q8. Prove: $\bigl||a|-|b|\bigr|\le|a+b|.$ When does equality hold?

Proof. Replace $b$ by $-b$ in the Reverse Triangle Inequality $\bigl||a|-|b|\bigr|\le|a-b|$: since $|-b|=|b|,$ we get $\bigl||a|-|b|\bigr|\le|a-(-b)|=|a+b|.$ $\blacksquare$

Equality: holds when $a$ and $b$ have opposite signs (or one is zero), i.e., $ab\le 0.$

Q9. Suppose $|x-2|<\dfrac{1}{2}.$ Show that $|x^2-4|<\dfrac{5}{2}.$

Solution. From $|x-2|<\tfrac12$: $\tfrac32<x<\tfrac52,$ so $x+2\in(\tfrac72,\tfrac92)$ and $|x+2|<\tfrac92.$
Then $|x^2-4|=|x-2||x+2|<\tfrac12\cdot\tfrac92=\tfrac94<\tfrac52.$ ✓

Q10. Prove: if $|x-a|<\varepsilon$ and $|y-b|<\varepsilon,$ then $|(x+y)-(a+b)|<2\varepsilon.$

Proof. $|(x+y)-(a+b)|=|(x-a)+(y-b)|\le|x-a|+|y-b|<\varepsilon+\varepsilon=2\varepsilon.$ $\blacksquare$

Q11. Let $|x|\le 1.$ Prove $|x^3-x^2|\le 2.$

Proof. $|x^3-x^2|=|x^2||x-1|\le|x|^2\cdot(|x|+1)\le 1\cdot(1+1)=2.$ $\blacksquare$

Q12. Determine all $x\in\mathbb{R}$ satisfying $\bigl||x-1|-2\bigr|=3.$

Solution. $||x-1|-2|=3\iff|x-1|-2=\pm 3.$
Case $+3$: $|x-1|=5\iff x-1=\pm5\iff x=6$ or $x=-4.$
Case $-3$: $|x-1|=-1,$ impossible ($|\cdot|\ge 0$).
Answer: $x\in\{-4,\,6\}.$

Q13. Verify that $\max\{a,b\}=\dfrac{a+b+|a-b|}{2}$ and $\min\{a,b\}=\dfrac{a+b-|a-b|}{2}$ by testing $a=5, b=2$ and $a=-1, b=3.$

Solution. Test $a=5, b=2$: $|a-b|=3.$ $\tfrac{5+2+3}{2}=5=\max\{5,2\}$ ✓   $\tfrac{5+2-3}{2}=2=\min\{5,2\}$ ✓

Test $a=-1, b=3$: $|a-b|=4.$ $\tfrac{-1+3+4}{2}=3=\max\{-1,3\}$ ✓   $\tfrac{-1+3-4}{2}=-1=\min\{-1,3\}$ ✓
